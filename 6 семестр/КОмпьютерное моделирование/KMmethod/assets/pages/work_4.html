Работа №4

Формирование графического интерфейса станции оператора

Цель работы

Создание графического интерфейса станции оператора.

Задание

Создать и настроить графический интерфейс для модели управления технологического процесса для станции оператора.

Создать проект в менеджере задач

Это работа завершающая. В результате неё у вас получится модель управления ТП, которая объединяет в себе все предыдущие работы.

В данной работе используются 2 приложения: Генератор динамики и Станция оператора. С помощью генератора динамики вы создаёте и настраиваете графический интерфейс. Через станцию оператора вы взаимодействуете с этим интерфейсом в роли оператора.

В Менеджере задач необходимо создать проект, к которому в дальнейшем подключим остальные составные модули. При создании указать имя и путь к БД, созданной в работе №2.

Для дальнейшей настройки проекта необходимо запустить созданный проект через генератор динамики и создать в нём проект графической базы данных.

Настроить проект

Для работы с Генератором динамики нужно указать рабочую директорию (расположение проекта). Для корректной работы модели в директории с проектом должны находиться файлы:

  • Папка с БД,
  • файл .kgp,
  • файл .kgx,
  • файл .kgp.gdi

В дополнительных свойствах проекта настройте параметры запуска для Генератора динамики и Графического интерфейса. Для этого укажите абсолютный путь до файла генератора динамики .kgp в кавычках (Рис. 1).

Конфигурация — вариант запуска проекта. Для данной работы достаточно 1 конфигурации.

В свойствах конфигурации укажите абсолютный адрес до файла генератора динамики .kgp в кавычках (Рис 1.)

Рис. 1. Настройка проекта

Создать мнемосхему

Мнемосхема — это визуализация вашей системы с динамическими элементами, состояние которых зависит от значений переменных. Например, лампочка, которая загорается, когда температура (ВА) падает ниже заданного значения (правила заданные в 3 работе).

В данной работе вам требуется создать минимум 3 мнемосхемы. Зачастую это:

  • Главная мнемосхема, отражающая работу систему,
  • Мнемосхема с трендами,
  • Мнемосхема с управлением — элементами переключения между мнемосхемами на рабочем столе (содержаться в базовой библиотеке).

Мнемосхемы составляются из геометрических примитивов, можно брать элементы из стандартной библиотеки. При необходимости возможно добавить изображение.

Для данной работы могут понадобиться индикаторы, кнопки, тренды, ёмкости, приборы и другие элементы. Для манипуляции с переменными ручного ввода используйте приборы соответствующего типа.

Подключить динамику к переменным

Настройка динамики заключается в привязывании переменных к элементам мнемосхемы (значений переменных к состояниям элементов).

Создать рабочий стол

Рабочий стол — пространство, в котором отображается совокупность мнемосхем, с которым будет взаимодействовать оператор.

В данной учебной задаче достаточно создать 1 рабочий стол. В настройках рабочего стола укажите его содержание (управляющая мнемосхема) и отметьте основной стол.

Назначить переходы

Чтобы элементы переходов работали, необходимо создать сами переходы и привязать к ним мнемосхемы. В настройках переходов укажите какая мнемосхема откроется и в каком месте рабочего стола будет расположена (Рис. 2).

Рис. 2. Настройка перехода

Проверить модель на работоспособность

Корректность работы системы можно проверить в режиме имитации в Генераторе динамики. Проверка рабочих столов проводится в Станции оператора.
