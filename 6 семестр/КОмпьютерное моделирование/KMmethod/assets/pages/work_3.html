Задание

Разработать алгоритм и написать программу для контроллера в интегрированной среде разработки КРУГОЛ, используя два инструмента: язык структурированного текста и язык функционально-блочных диаграмм.

Создать проект в ИСР КРУГОЛ

КРУГОЛ — интегрированная среда разработки. В данной работе мы будем использовать КРУГОЛ для реализации правил поведения переменных, разработанных в лабораторной работе №1.

Перед началом написания программы в ИСР КРУГОЛ нам необходимо создать проект. Он будет объединять наши файлы и связывать программу с БД.

Создать файл ФБД или СТ

Для реализации правил поведения переменных, разработанных в лабораторной работе №1 в КРУГОЛ заложены два инструмента на выбор: языки структурного текста (СТ) и функциональных блочных диаграмм (ФБД).

СТ представляет собой язык скриптов со своим синтаксисом. Для учебной задачи использование СТ ограничится написанием простейших конструкций «Если - то». Однако при необходимости возможна реализация и более сложных конструкций: циклы, процедуры и другие.

ЕСЛИ вд3 = 1 { :система ароматизации
дв4 = 1
}

ИНАЧЕ {
дв4 = 0
}
ЕСЛИ ва7 > 10 { :уровень хладагента
дв3 = 1
дв1=1
дв5=1 :работа вентиляторов
}

Язык ФБД — графический. Работа с ним заключается в составлении графической схемы из блоков встроенной библиотеки. В рамках нашего задания нам понадобятся блоки простейших функций: арифметические операции, операции сравнения. Результат получается более наглядным, но достаточно громоздким даже для небольшого учебного проекта.
