3.2.1. Запустить исполняемый файл Tact_1804.exe.

 После успешного запуска будет отображено основное окно программы, представленное на рисунке 1, которое остается активным в течение всего времени ее функционирования.

 Рисунок 1 Основное окно


 3.2.2. Основное окно содержит набор меню, при выборе конкретного меню раскрывается диалоговое окно установки тех или иных параметров или выполнения определенных действий:

 -          Файл –  используется для создания нового или открытия существующего файлов.

                      *Создать –  используется для создания нового файла.

                      *Открыть  –  используется для выбора существующего файла.

                      *Сохранить как – используется для сохранения измененного , или созданного файлов. 

                      *Выход - используется для выхода из программы.

 -          Настройки –  содержит параметры программы.

                      *Задать объем PZU –  используется для указания объема PZU.

                      *Указать вариант –  используется для указания варианта.

 -          Справка – содержит справочную информацию о программе.

                       *О программе –  содержит информацию о создателях, дату выпуска программы и другие сведения. .

                       *Файл справки –  краткое описание программы.
