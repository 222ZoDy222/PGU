2.1. Для работы программы необходимы:

  • тактовая частота процессора 1 ГГц;
  • оперативная память ОЗУ емкостью 512 Мбайт;
  • стандартная клавиатура;
  • манипулятор (типа "мышь");
  • монитор с разрешением 640*480 точек ( минимум);  

2.2. Программа функционирует  в операционной среде Windows XP, 7, 8.

2.3. Программа разработана на языке С++ и  оформлена в виде:

  • загрузочного модуля Tact_1804.exe;
  • библиотек msvcr110.dll, msvcp110.dll.
  • файла помощи help_tact.chm

2.4. Для работы программы необходим один из дистрибутива пакета Microsoft .NET Framework 3.5 , который устанавливает среду .NET Framework и связанные с ней файлы, необходимые для запуска приложений, созданных для работы в среде выполнения .NET Framework 3.5.

2.5. Конечный пользователь программы  должен иметь среднее образование, обладать практическими навыками работы с графическим интерфейсом операционной системы.
